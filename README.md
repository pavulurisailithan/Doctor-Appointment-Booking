# 🩺 MERN Stack Doctor Appointment Booking App

The **MERN Stack Doctor Appointment Booking App** is a full-stack web application developed using **MongoDB, Express.js, React.js, and Node.js**. It provides a seamless platform for patients to book appointments with doctors online while enabling doctors and administrators to efficiently manage schedules, appointments, and user information.

The application offers a user-friendly interface with secure authentication, real-time appointment management, and role-based access for patients, doctors, and administrators. It is designed to simplify the healthcare appointment process by reducing manual scheduling and improving communication between patients and healthcare providers.

## 🚀 Features

### 👤 Patient
- User Registration and Secure Login
- Browse Doctors by Specialization
- View Doctor Profiles
- Book Doctor Appointments
- View and Cancel Appointments
- Update Personal Profile

### 👨‍⚕️ Doctor
- Secure Login
- Manage Profile Information
- View Scheduled Appointments
- Accept or Reject Appointment Requests
- Update Appointment Status
- View Patient Details

### 👨‍💼 Admin
- Secure Admin Dashboard
- Manage Doctors
- Manage Patients
- Monitor All Appointments
- Add or Remove Doctors
- View System Statistics
